About:
	This is a very simple PHP router.  The purpose of this router was
	to create something simple and light weight for web services.  
	Everything here is just a skeleton router.  You'll need to create
	any extra features yourself.  
	
Directory Structure:

	public
		-> .htaccess
		-> index.php
	src
		-> route.php
		-> home.php
		-> about.php
		-> contact.php

		
How To Use:
	Edit .htaccess as necessary.  
	
	Create classes in the src directory and then add the routes in the 
	route.php file. There are some basic classes included as reference 
	
	Enjoy!
